Grid is a 2-Dimensional layout system. It can layout entire sites or smaller components. It allows for precise item placement within containers. 
display: grid
 sets a grid for the container used such as divs, main, sidebar, and others. These contain elements on the page, determines how they are laid out inside the container. Only operates on direct children elements. Governed by rows and columns.
